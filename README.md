# HistoraGraph-AI

AI-powered Historical Knowledge Graph Explorer.

## Overview

HistoraGraph-AI is a knowledge graph based AI system
that allows users to explore historical knowledge,
entities, events and relationships using Graph Database
and Retrieval Augmented Generation.

## Main Features

- Historical Knowledge Graph
- Entity Relationship Exploration
- AI Question Answering
- RAG-based Historical Assistant
- Timeline Visualization

## Technologies

Backend:
- FastAPI
- Python

Frontend:
- ReactJS

Database:
- PostgreSQL
- Neo4j
- Qdrant

AI:
- LangChain
- LLM API
- Embedding Model
